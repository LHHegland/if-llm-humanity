This is the master map of the entire knowledge system. Use it to quickly find

# Humanity Wisdom — Master Index

**Purpose:** Navigation guide for all humanity-wisdom reference files. Maps
plain-language concepts to their file location, ID, and canonical tag. Use this
to quickly orient to the right reference file before diving into content.

**How to use:**
1. Identify the closest concept to what the person is asking about.
2. Note the file and the canonical tag.
3. Read only the relevant section of that file — you don't need to load everything.
4. If an exact concept isn't listed, use the nearest parent concept.
5. If something isn't found at all, say so explicitly, then continue with
   general knowledge labeled as such.

**Tag format:** `[[HUMANITY:<LEAF>]]` — for example, `[[HUMANITY:WISDOM]]`

## Supported Use Cases (Plain-English)

This knowledge system is designed for situations like:

- "I keep getting into the same argument with my partner. Why does this happen?"
- "I don't understand why some people believe things that seem obviously wrong."
- "I'm worried about losing my job to AI. What should I do?"
- "How do I talk to my kids about what's happening in the news?"
- "I feel like society is falling apart. Is that normal? Has it happened before?"
- "I want to be a better person. Where do I even start?"
- "Why do people keep repeating history's mistakes?"
- "My neighbor is struggling and I want to help, but I don't know how."
- "I'm trying to make a hard decision and I keep going in circles."
- "What do all the world's wisdom traditions actually agree on?"

**Not designed for:** diagnosis, therapy, legal advice, financial planning,
technical expertise, or emergency response. For those, refer to professionals.
See `01-framework.md` → Coping for crisis resources.
